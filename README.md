# AI-Powered E-Commerce Web App

A full-featured web application combining traditional e-commerce with the power of AI, delivering a smart and seamless shopping experience.

## Overview

This project enables users to generate personalized meal plans and budgets using the OpenAI API. Users can instantly convert their plan into a shopping cart with a single click, making shopping intelligent and effortless.

## Key Features

- **Modern Infrastructure:** Built with Next.js 16 (App Router & Server Actions) and TypeScript for high performance and scalability.  
- **Artificial Intelligence:** Integrated OpenAI API to generate personalized plans based on user inputs and budget.  
- **Payment System:** Full Stripe integration with Webhooks to secure transactions and automatically update inventory.  
- **Dashboard & Analytics:** Dynamic dashboard built with Recharts to display revenue, orders, and active users in real-time.  
- **Database Management:** Powered by Supabase for reliable and efficient data handling.

## Tech Stack

- **Front-end & Framework:** Next.js 16, React, TypeScript  
- **Back-end & Database:** Supabase (PostgreSQL)  
- **AI:** OpenAI API  
- **Payments:** Stripe & Webhooks  
- **Analytics:** Recharts
